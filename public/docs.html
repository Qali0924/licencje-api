Integracja API

Zabezpiecz swój projekt w kilka minut korzystając z globalnej infrastruktury Synthesized Studio.

1

Wprowadzenie

System walidacji opiera się na żądaniach HTTP. Plugin przesyła klucz oraz IP serwera, a nasze API weryfikuje dane w czasie rzeczywistym. W przypadku braku aktywnej licencji, zalecamy natychmiastowe zatrzymanie logiki pluginu.
2

Endpoint Walidacji

Metoda HTTP

POST

Endpoint URL

https://licencje-api-u0sq.onrender.com/api/validate

Struktura JSON (Request)

{
  "key": "TWOJ-KLUCZ-LICENCYJNY",
  "ip": "AUTO"
}
3

Implementacja Java (Bukkit/Spigot)

Zależność GSON (pom.xml) 

<dependency>
    <groupId>com.google.code.gson</groupId>
    <artifactId>gson</artifactId>
    <version>2.10.1</version>
</dependency>
