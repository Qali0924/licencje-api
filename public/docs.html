Integracja API

Dowiedz się, jak w kilku prostych krokach zabezpieczyć swój plugin Minecraft korzystając z systemu CLOUD MATH.

1

Wprowadzenie

Plugin przy starcie serwera wysyła zapytanie typu POST do naszego systemu. Jeśli klucz jest poprawny, a limit IP nie został przekroczony, API zwraca status pozytywny. W przypadku braku licencji, zalecane jest wyłączenie pluginu (disablePlugin).
2

Endpoint API

Metoda

POST

URL Walidacji

https://twoja-domena.pl/api/validate

Przykład zapytania (JSON)

{
  "key": "TWÓJ_KLUCZ_LICENCYJNY",
  "ip": "IP_SERWERA_MINECRAFT"
}
